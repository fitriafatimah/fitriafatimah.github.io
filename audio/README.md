# Folder Audio

Simpan file musik Anda di folder ini.

## Format yang Didukung:
- MP3 (`.mp3`) - **Direkomendasikan**
- WAV (`.wav`)
- OGG (`.ogg`)

## Contoh:
- `music.mp3`
- `background-music.mp3`
- `portfolio-theme.mp3`

## Cara Menggunakan:
1. Simpan file musik Anda di folder ini
2. Update `js/data.js` dengan path file musik:
   ```javascript
   musicUrl: "audio/music.mp3"
   ```

Lihat `TUTORIAL-MUSIK.md` di root project untuk panduan lengkap.
